# 촉각 칼라 북 — Tactile Color Book Viewer

성균관대학교 휴먼ICT융합학과(조준동 교수)에서 제작한 「촉각 칼라 북: 시각장애인들의 칼라 이해를 돕는 촉각 명화 감상」을 웹에서 페이지별로 넘겨보며 감상할 수 있는 HTML 뷰어입니다.

🔗 **데모:** `https://jdcho0721.github.io/tactile-color-book/`

## 미리보기

- 목차 패널 — 1장 칼라 기초 이론 · 2장 칼라 패턴 · 3장 칼라와 형태를 촉각화한 명화 작품 5선(반 고흐, 피카소, 마티스, 프리다 칼로, 고갱)으로 구분되어 원하는 항목을 클릭하면 해당 페이지로 즉시 이동합니다.
- 하단 필름스트립 — 전체 68페이지 썸네일을 한 줄로 훑어보며 클릭 이동할 수 있습니다.
- 키보드 `←` `→` 페이지 이동, `T` 목차 열기, 모바일 스와이프 지원.
- 페이지 이미지를 클릭하면 확대해서 세부 패턴을 자세히 볼 수 있습니다.

## 기술 구성

순수 HTML · CSS · JavaScript로만 작성되어 별도의 빌드 과정이나 서버가 필요 없습니다. 책의 각 페이지는 미리 렌더링된 JPEG 이미지(`pages/page-01.jpg` ~ `page-68.jpg`)이며, `index.html`이 목차·내비게이션·확대 기능을 제공합니다.

## 로컬에서 보기

이미지 기반 정적 페이지라 별도 서버 없이 `index.html`을 더블클릭해서 바로 열 수 있습니다.

## GitHub Pages로 배포하기

1. 이 저장소의 **Settings → Pages**로 이동
2. **Build and deployment → Source**를 `Deploy from a branch`로 설정
3. Branch를 `main` / `(root)`로 선택 후 저장
4. 1~2분 뒤 `https://jdcho0721.github.io/tactile-color-book/`에서 확인 가능

## 디렉터리 구조
